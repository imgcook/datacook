# datacook
Library for data processsing in JS

## Developers

Fork the repo, and then clone to your local machine

```
git clone https://github.com/imgcook/datacook.git
cd datacook
```
Install dependencies

```
yarn
```
or if you use npm

```
npm install
```

Run test
```
yarn test
```

or if you use npm

```
npm run test
```
