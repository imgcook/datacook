# shuffle()

Randomly exchange the element positions in an array.

## Syntax

```typescript
shuffle(inputs: Array<any>): void
```

## Parameters

| Parameter |        type        | description                                                         |
| :-------- | :-----------------: | :------------------------------------------------------------------ |
| inputs | Array<any> | Array of objects for shuffle |
